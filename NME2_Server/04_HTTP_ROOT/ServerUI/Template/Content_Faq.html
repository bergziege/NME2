FAQ

What is NME2?

NME2 stands for "Network Mission Engine 2".
A few tools that let you place scenery objects over the internet.
As an example you can build an air race course nobody had ever seen before and just activate it for an event.

What are those tools?

1. The client running on the flight simulator PC.
2. This server interface for a quick overview of all missions available on the server
3. A management program to administrate the missions on the server

Any requirements?

Client: Requires FSX over a simconnect connection, .NET 3.5 and an active internet connection.
Manager: Requires .NET 4.0 and an active internet connection.

Where to get the tools?

The client and manager files can be downloaded in die download section.
The Server will be available for download later this year.

How it is all working together?

1. Creating a mission with the NME2 Manager, adding some objects and save it to the server.
2. Connect your client to the server.
3. Ask the server for new scenery items to sync before connecting to FSX
4. Connecting the client to FSX.
5. Fly as normal.
6. In the background the client is monitoring your position and asking the server if your current position lies within any mission.
7. If you are currently within a mission the client gets the information about the mission objects and activates them inside FSX.
8. After leaving a mission the items will be deactivated.
 
The interval to check for new missions can be set freely within the client.
